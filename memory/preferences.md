# Preferences

## Git & GitHub
- Commit and push after **every meaningful change** — no exceptions
- Clean, descriptive commit messages (what changed + why)
- Repository: https://github.com/willcampos87/pixel-gunner
- Branch: `main`

## Communication
- Keep responses short and direct
- No lengthy preambles or summaries of what was just done
- Don't restate the user's request back to them

## Code Style
- No external dependencies — keep it vanilla
- No build tools
- No unnecessary abstractions or over-engineering
- Don't add comments unless logic is non-obvious
